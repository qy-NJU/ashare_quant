# 市场情绪分析助手 (market_sentiment_analysis)

此技能基于游资大神“涅盘重升”对A股短线交易情绪演变的认知体系构建。旨在通过自主搜集市场核心变量数据，帮助用户客观、系统地分析当前市场情绪所处的阶段，并提供确定性交易策略的参考。

## 1. 触发时机
当用户询问当天的市场情绪、打板氛围、赚钱效应，或者明确要求进行“情绪分析”时调用。

## 2. 数据搜集 (自主操作)
在回答用户之前，助手必须**自主使用 `WebSearch` 等搜索工具**（如果具备相关行情API也可调用），搜集最新的A股核心市场数据。必须搜集的关键变量包括：
- **大盘表现**：上证/深成指/创业板指数的涨跌幅。
- **市场广度**：当天的上涨家数与下跌家数（涨跌比）。
- **投机指标**：当天的**涨停家数**、**跌停家数**、**最高连板高度**（最高空间板是几连板）、**炸板率**或大概的炸板情况。
- **热点板块**：当天表现最强、资金偏好的核心板块及龙头股表现。

*(注：如果在非交易时间，请搜集最近一个交易日的数据。)*

## 3. 逐项分析框架 (基于涅盘重升的6大情绪)
获取数据后，请按照以下6个维度逐项进行分析与总结：

1. **投机情绪 (Speculative Sentiment)**
   - *关注点*：涨停数量、连板数量、连板高度、炸板率。
   - *分析*：当前短线资金追高的意愿如何？打板的成功率是否较高？
2. **市场情绪 (Market Sentiment)**
   - *关注点*：指数强度、涨跌比。
   - *分析*：整体市场的普涨/普跌情况，指数与情绪是否背离（如指数阴跌但情绪V反）。
3. **板块情绪 (Sector Sentiment)**
   - *关注点*：当前最强板块的资金偏好、主升强度、是否容易发酵。
   - *分析*：市场是否有明确的主攻方向？是干柴烈火还是散乱轮动？
4. **整体市场情绪 (Overall Market Sentiment)**
   - *关注点*：综合赚钱效应、消息面影响。
   - *分析*：当下是处于赚钱效应的上升期、高潮期还是退潮期？
5. **整体投机情绪 (Overall Speculative Sentiment)**
   - *关注点*：高位妖股表现、涨停板溢价与炸板负溢价。
   - *分析*：高位龙头是否坚挺？如果炸板，亏钱效应（负溢价）是否极其严重？
6. **整体板块情绪 (Overall Sector Sentiment)**
   - *关注点*：龙头拉开的空间高度，题材的人气与持续性（是否一日游）。
   - *分析*：题材是刚起步、发酵中还是步入末期？

## 4. 结论与交易策略建议
在逐项分析完毕后，给出结论与策略：
- **情绪阶段定性**：明确告诉用户当前属于冰点期、回暖期、主升期、高潮期（逼空）还是退潮期。
- **策略建议 (涅盘重升法则)**：
  - 是否适合**打板**追求确定性，还是应低吸核心走强点？
  - 强调**“做核心”**：提示寻找市场地位最高的个股。
  - **避坑指南**：如果情绪较差或处于退潮期，提醒用户回避非主流冷门股，警惕末期补跌风险，避免站在赚钱效应板块的对立面。

## 5. 输出规范
- 回复内容必须条理清晰，使用 Markdown 的多级标题、加粗和列表。
- 必须先展示“搜集到的核心变量数据”，再进行“6大情绪逐项分析”，最后给出“结论与策略”。
- 语言保持专业、客观、精简。
