# Product Design

This skill helps users design product outlines based on simple input. It gathers necessary information, thinks globally to plan the project structure, and clarifies doubts with the user.

## Capabilities

1.  **Information Gathering**: Spontaneously collect necessary information based on user's simple input.
2.  **Global Planning**: Think from a global perspective to plan the general outline of the project.
3.  **Clarification**: Identify ambiguous or doubtful areas and confirm them with the user.

## Instructions

When this skill is invoked:

1.  **Analyze User Input**: Understand the core idea provided by the user.
2.  **Research & Expansion**:
    *   Use search tools to understand the domain if necessary.
    *   Identify key features, user roles, and technical requirements.
3.  **Draft Outline**: Create a high-level project plan including:
    *   Project Goal
    *   Key Features (MVP vs Future)
    *   Tech Stack Recommendation
    *   Architecture Overview
4.  **Identify Doubts**: List questions for the user regarding specific requirements, constraints, or preferences.
5.  **Interact**: Present the outline and questions to the user for feedback.
